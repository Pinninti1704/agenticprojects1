Which frontend stack?

You mentioned "frontend plugins" — what's your preference?

A

React + Vite + Tailwind CSS

Modern, component-based. Tailwind for utility-first styling. Recharts for charts. Lucide for icons. Same stack as the trading dashboard, so you already have it installed. No backend needed — pure static SPA with localStorage.

B

React + Vite + shadcn/ui

Beautiful, accessible component library built on Radix UI + Tailwind. Professional look out of the box. Slightly heavier setup but gives you polished tables, dialogs, forms, and charts with minimal custom CSS.

C

React + Vite + MUI (Material UI)

Full component library with Google Material Design. Lots of built-in components (data grids, autocompletes, steppers). Heavier bundle but feature-rich. Well-suited for data-heavy dashboards.

D

Vanilla HTML/CSS/JS

No framework, no build step. Keep it lean. All styling custom. Maximum control, minimum dependencies. Good for a focused single-page tool with no complex state management needs.
