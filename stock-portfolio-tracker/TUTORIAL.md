# Stock Portfolio Tracker - Quick Start Tutorial


## 1-Minute Setup


```bash
cd stock-portfolio-tracker
npm install
npm run dev
```

Then open: **http://localhost:5174/**


## 1-Minute Guide


### Adding Your First Stock


1. In the "Stock Symbol" field, type: **AAPL**
2. The suggestion "AAPL" will appear - click it
3. Enter **10** for shares
4. Enter **150.25** for purchase price
5. Click **"Add Stock"** button

✅ You now have Apple (AAPL) tracked in your portfolio!


### Adding Your Second Stock


1. Type: **MSFT**
2. Enter **5** shares
3. Enter **300.50** purchase price
4. Click **"Add Stock"**


### Viewing Your Portfolio


Your portfolio now shows:
- **Total Cost Basis**: How much you invested ($10×$150.25 + $5×$300.50 = $3006.25)
- **Current Value**: Real-time mock prices show your profit/loss
- **Gain/Loss**: How much money you've made or lost
- **List of Holdings**: Shows each stock with current value and percentage change


### Selling a Stock


If you want to remove a stock, just click the **✕** (remove) button next to it!


## Pro Tips


- You can add more shares to an existing stock by entering the same symbol again
- The app shows green for gains and red for losses
- All calculations are automatic - no manual math needed!
- Works on mobile phones too
